# microservices with nodejs

A repository to store my progress on the course [Microservices with Node JS and React](https://www.udemy.com/course/microservices-with-node-js-and-react)

## Tech Stack

**Client:** React

**Server:** Node.js, Express.js, MongoDB, Docker, Kubernetes, NATS Streaming Server, Skaffold, Ingress-Nginx
